# george-cxue.github.io

Personal portfolio and website built with Next.js.

## Stack

- **Next.js 16** — App Router, static export
- **React 19** — with view transitions
- **TypeScript**
- **Tailwind CSS**
- **Markdown** — via unified/remark/rehype pipeline

## Sections

- **Thoughts** — blog posts written in markdown
- **Projects** — selected work
- **Photos** — photo exhibits with swipe navigation
- **People** — people I admire
- **Quotes** — quotes I find interesting

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is statically exported to `/out` and deployed via GitHub Pages.
